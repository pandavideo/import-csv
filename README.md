# PANDA IMPORT CSV

## Usage:
Instalar pacotes necessários:
```sh
npm install
```

Popular arquivo `.env` com base no `.env.example`

Para importar os vídeos:
```sh
node upload.js
```
Para verificar os imports:
```sh
node verify.js
```

> **ATENÇÃO:** verificar se campos do CSV são correspondentes e mudar caso necessário
>
